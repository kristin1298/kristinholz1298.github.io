Digital Waitingroom
July 2020
The "Digital Waitingroom Project" is an application that makes it easier for users to book a doctor's appointment. With the help of a few clicks, they can choose a suitable appointment and also set a reminder. Therefore, the user is reminded in time of his appointment and, if necessary, also informed if delays may occur. In this way, the user does not have to wait in the medical practice and can spend the waiting time reasonably.

The Problem

Overcrowded waiting rooms have always
carried a high risk of infection. Especially
in times of corona this must be avoided
and so the risk of infection for all people
must be reduced.
But how could this Problem be solved?

The Solution

You need an application that allows users
to book a doctor's appointment easily and
spontaneously. It must also be possible for
the user to be informed in time if there are
delays. This way the user does not have to
wait in the clinic, but can use the waiting
time reasonably.
